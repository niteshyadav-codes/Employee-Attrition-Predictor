# Employee Attrition Predictor

## 📌 Project Overview

Employee Attrition Predictor is a machine learning project that predicts whether an employee is likely to leave the company or not. The model is trained on the IBM HR Analytics dataset and analyzes factors such as age, salary, department, overtime, job satisfaction, and work-life balance.

This project compares multiple machine learning algorithms and selects the best-performing model based on accuracy.

---

## 🎯 Objective

The main goal of this project is to:

* Predict employee attrition.
* Analyze the factors affecting employee turnover.
* Compare different machine learning algorithms.
* Build a professional machine learning pipeline.

---

## 📂 Dataset

**Dataset:** IBM HR Analytics Employee Attrition Dataset

The dataset contains employee information such as:

* Age
* Department
* Education
* Job Role
* Gender
* Monthly Income
* Overtime
* Work-Life Balance
* Job Satisfaction
* Years at Company

**Target Variable:**

* `Attrition`

  * Yes → Employee will leave the company.
  * No → Employee will stay.
